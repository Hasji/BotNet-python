Этот ботнет состоит из 2 основных файлов: Server.py (это HTTP-сервер, который содержит API, панель C&C, базы данных sqlite и сеанс) и файл Core.py, который является агентом, с помощью которого заражаются боты.

Выполните следующую команду: py db.py init
Отредактируйте все глобальные переменные с вашей конфигурацией (внутри Server.py), такие как HOST, DB_PATH, API и другие.
Запустите сервер, используя следующую команду на сервере: py Server.py
Перейдите на локальный хост: 8080.
Вход под юзером, паролем* и ключом ботмастера
Запустите Core.py на машине-жертве с помощью следующей команды: py Core.py
Подождите несколько секунд, пока агент отправит нам всю информацию
Исследуйте панель C&C !!! (проверьте таблицу команд)
Pass = пароль будет сгенерирован автоматически с использованием кодировки AES и base64, поэтому вы должны копировать его каждый раз, когда запускаете сервер, потому что он действителен один раз.
Учетные данные можно получить только из Chrome, и они будут сохранены в таблице Creds БД creds.db 
После этого вы можете увидеть их для каждого отдельного бота. Если вы хотите получить учетные данные, вы вставите в качестве команды для выполнения специальное ключевое слово cred, которое запускает модуль Cred в агенте.
